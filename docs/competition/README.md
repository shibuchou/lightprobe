# 官方赛题资料

本目录将 2026 年操作系统设计赛 OS 功能挑战赛道当前可核验的官方要求整理为可直接阅读的中文文字文档。

## 文档目录

1. `official_competition_rules.md`
   - 赛事、赛道、参赛资格、组队规则、指导教师规则、报名流程、队员调整规则和时间节点。
2. `registration_and_submission_materials.md`
   - 报名表字段、身份证明材料、电子版和扫描件提交要求。
3. `open_source_and_integrity_commitment.md`
   - 源代码、文档、仓库公开状态、知识产权和学术诚信要求。
4. `official_scoring_rules.md`
   - `proj40` 官方评分细则的当前获取状态，以及哪些项目内容不能冒充官方评分标准。
5. `competition_compliance_checklist.md`
   - 根据上述官方要求整理的项目执行检查表，方便提交前逐项核对。
6. `official_metadata.json`
   - 2026-06-14 获取的结构化赛道元数据快照。
7. `original/`
   - 后续存放从赛事官网下载的任务书、评分表和通知原件。
